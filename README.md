# idr4000-eksamensoppgave2

Dette repositoriet inneholder analysefiler og data til **Eksamensoppgave2 i emnet *IDR4000 - Kvantitativ metode og statistikk*.

## Innhold

**Data/**
Inneholder datasettet brukt i analysene (`hypertrophy.csv`).

**Analyser/**
Inneholder R-script (`analysis.R`) som utfører alle statistiske analyser og genererer figurer.

**Figurer/**
Inneholder figurer brukt i rapporten (histogram og spredningsdiagram med regresjonslinje).

**Dokumentasjon/**
Inneholder tabeller og støttedokumentasjon brukt i rapporten.

##Reproduserbarhet

Alle analysene er gjennomført i R. Scriptet i mappen **Analyser/** kan kjøres for å reprodusere resultatene presentert i rapporten.
